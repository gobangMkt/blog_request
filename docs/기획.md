# 네이버 블로그 제작 요청 앱 기획

## 개요

지점별 네이버 블로그 상위노출 작업을 효율적으로 관리하기 위한 신청 앱.
요청자가 키워드·지점·강조 내용을 입력하면, 구글 스프레드시트에 자동 저장되고, 작업 완료 후 카카오 알림톡으로 결과를 전달하는 흐름.

---

## 핵심 플로우

```
요청자 입력 → 키워드 중복/재사용 검증 → 구글 시트 저장 → 블로그 작업 → 결과 기록 → 카카오 알림톡 발송
```

---

## 기능 요구사항

### 1. 신청 폼 (프론트엔드)

| 필드 | 설명 |
|------|------|
| 이름 | 필수 |
| 전화번호 | 필수. 알림톡 발송 대상 번호 (`010-XXXX-XXXX` 형식 강제) |
| 키워드 1~3 | 각각 별도 입력칸. 입력 즉시 사용 가능 여부 실시간 피드백 |
| 지점 URL | `https://gobang.kr/place/[숫자]` 형식 강제 검증 |
| 강조 내용 및 설명 | 자유 텍스트 (textarea) |
| 제출 버튼 | 모든 유효성 검사 통과 시 활성화 |

### 2. 키워드 유효성 검사 (핵심)

- 구글 시트의 **완료 내역** 시트를 기준으로 판단
- 동일 키워드가 **3달(90일) 이내**에 상위노출 완료된 이력이 있으면 → 사용 불가
- 입력 시 즉시(onChange) 검사하여 피드백 표시:
  - 사용 가능 → ✅ 초록색 "사용 가능한 키워드입니다"
  - 사용 불가 → ❌ "이 키워드는 N일 뒤 사용 가능합니다. 다른 키워드를 선택해 주세요."

### 3. 지점 URL 유효성 검사

- 정규식: `^https://gobang\.kr/place/\d+$`
- 형식 불일치 시 제출 버튼 비활성화 + 안내 메시지 표시

### 4. 접근 권한 관리

- 기본: 링크를 가진 누구나 접근 가능
- 관리자가 구글 시트 **설정** 시트의 `앱 활성화` 셀을 `ON` / `OFF`로 변경하여 즉시 수신 제어
- `OFF` 상태에서 접근 시 "현재 신청을 받지 않습니다" 안내 화면 표시

## 키워드 검증 로직

```
입력된 키워드 → GAS API 호출
  → "완료 내역" 시트에서 해당 키워드 검색
  → 완료일이 오늘 기준 90일 이내인 행이 있는가?
    YES → 완료일 + 90일 - 오늘 = 남은 일수 반환 → "N일 뒤 사용 가능"
    NO  → 사용 가능 반환
```

## 결과 전달 방식

### 현재 방식 (수동)
- 알림톡 템플릿 승인 대기 중이므로, 관리자가 시트에서 전화번호를 확인하여 솔라피에서 수동 발송
- 시트의 `신청 내역`에 이름 + 전화번호가 쌓이므로 별도 수집 불필요

### 추후 자동화 (템플릿 승인 후)
- GAS 트리거 → 솔라피 API 호출 → 알림톡 자동 발송으로 전환 예정
- 알림톡 메시지 템플릿 (예시):
  ```
  [고방] 블로그 작성 완료 안내

  안녕하세요, {이름}님!
  요청하신 블로그 작성이 완료되었습니다.

  📍 지점: {지점URL}
  🔑 상위노출 키워드: {키워드}
  📝 블로그 링크: {블로그URL}

  감사합니다.
  ```
